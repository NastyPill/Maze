Select file to read/write - you should select a
.txt file on you PC which should be formatted properly
(it should have one entrance, one exit, at least one hole)

Seed text area - you can enter seed (number) for maze.

Size text area - you can enter size like:
 *width*x*height* size should be in 15-150

Click Start button to start

Use arrow keys to move.

You should pass through the maze to red cell.

In the maze:
. - hole
+ - wall
* - entrance
X - exit
